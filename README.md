# Next.js Project Skeleton

This project is a Next.js application initialized with TypeScript, Tailwind CSS, Shadcn UI, and Lucide Icons, following the `nextjs-project-architecture.md` specification.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v3
- **UI Components**: Shadcn UI (Stone Base / Blue Primary)
- **Icons**: Lucide React

## Getting Started

First, install the dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/ui/`: Shadcn UI components.
- `components/layout/`: Layout components.
- `components/shared/`: Shared components.
- `lib/`: Utility functions and types.
- `public/`: Static assets.
